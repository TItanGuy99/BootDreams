BootDreams

BootDreams is a tool for Windows that uses many command-line tools to create selfbooting images and discs for Sega Dreamcast.

It uses cdi4dc to create selfbooting DiscJuggler images, cdi2nero to convert DiscJuggler images into Nero images, mds4dc to create selfbooting Alcohol 120% images and cdrecord to burn selfbooting CDs and burn ISO, CUE, and most DiscJuggler images.
