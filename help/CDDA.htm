CDDA

BootDreams is capable of creating audio/data Alcohol 120% images, audio/data and data/data selfbooting CDs with CDDA. If "Add CDDA tracks" is checked (and enabled) in the Extras menu, a small window will display as the imaging/burning process begins and list all the supported audio formats in the folder cdda, allowing you to configure and organize what audio files you want as CDDA. If any audio file is not a raw PCM audio file, BootDreams will run SoX to convert it to raw PCM audio data for you.

If a data/data selfbooting CD is burned, the first track is data, not audio. You will need to take this into account if you want the right CDDA tracks played during the game.

Supported Audio Formats:

  • NeXT Sound (*.snd)
  • MPEG Audio Stream, Layer II (*.mp2)
  • MPEG Audio Stream, Layer III (*.mp3)
  • Ogg Vorbis Compressed Multimedia File (*.ogg)
  • Audio Interchange File (*.aif; *.aiff)
  • uLaw/AU Audio File (*.au)
  • Waveform Audio (*.wav)
  • Raw Signed PCM Data (*.raw)
  • Creative Labs Sound (*.voc)

Note: Some Dreamcast games require the CDDA to start at a track other than the first track. If you're burning commercial games this is normally 3 blank tracks. Other software, such as the Quake port "nxMakaqu", require 2 blank tracks.
