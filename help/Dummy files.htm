Dummy files

BootDreams is capable of creating audio dummy files and data dummy files using newfile. The space remaining to the disc is calculated and a dummy file is created to fill that space.

Audio dummy files are created in the following situations:

  • An audio\data Alcohol 120% image without CDDA
  • An audio\data selfbooting CD is burned without CDDA

In all other situations (data\data, discs with CDDA, DiscJuggler and Nero images), a data dummy (named 000DUMMY.DAT) is created.
