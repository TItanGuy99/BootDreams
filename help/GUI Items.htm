GUI Items

Toolbar

In order from left to right: DiscJuggler, Nero, Alcohol 120%, CDRecord, ImgRecord.

Browse button

Opens the folder or file open dialog depending on the current task.

Selfboot folder/CD image
 
Pretty self-explanatory. These text boxes display the path to the selected folder or file.

CD label

This is the label that displays in Windows in "My Computer." Maximum of 31 characters.

Disc format

This combo box sets the format of the CD image or disc-to-burn. If the current task is CDRecord, you can also select "Multisession."

Image format

Sets the format of the Nero image. See F.A.Q. for the differences.

Burn speed

Again, pretty self-explanatory but if your drive doesn't support the speed you set then the closest supported speed is used. The speed also depends on whether the media supports this speed.

Merge previous session

This allows you to keep what is already on a multisession disc visable. Otherwise the disc will have just the files you put on it this session.

Multisession

This allows you to burn ISO and CUE sheets in ImgRecord as a multisession CD. Make sure to remember to keep the LBAs in mind if adding to an already multisession CD.

Process button

This will start the image creation or CD burning process.
