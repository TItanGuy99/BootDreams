Menu Items

File:

Check for Update
Checks a file online and compares the version online with the version on your computer.

Exit
Exits BootDreams.

 

Extras:

ISO settings

  • Write Mode
    Sets the write mode in CDRecord multisession and when burning ISO images in ImgRecord.
    • Mode 1 (-data)
      Normal mode for everyday data burns. DemoMenu and DCHakker will read Mode 1 discs.
    • Mode 2 Form 1 (-xa)
      The format Dreamcast discs are burn in. Useful to switch to this mode if you made a coaster and want to burn a selfbooting CD.
  • Full filenames (-l)
    If checked, it allows full 31 character filenames in the ISO rather than 8+3 characters.
  • Joliet (-J)
    If checked, it generates Joliet directory information in the ISO. This allows filenames to be lower case and enable Windows to correctly display the filenames.
  • Rock Ridge (-r)
    If checked, it generates Rock Ridge directory information in the ISO. This allows filenames to be lower case and enable Unix environments (Dreamcast included) to correctly display the filenames.

Add CDDA tracks
If checked, a small window will display as the imaging/burning process begins and list all the supported audio formats in the folder cdda, allowing you to configure and organize what audio files you want as CDDA. If any audio file is not a raw PCM audio file, BootDreams will run SoX to convert it to raw PCM audio data for you.

Dummy file
If checked, BootDreams will generate either a audio dummy file or a data dummy file to fill the remaining free space available to a disc.

  • 650 MB CD-R
    Select this option if you have a 650 MB CD-R.

  • 700 MB CD-R
    Select this option if you have a 700 MB CD-R.

  • None
    If this option is selected no dummy file will be created.

Insert MR logo
If checked, a small window will display as the imaging/burning process begins and list the .mr files in the folder iplogos, allowing you to choose which MR logo to inject into the IP.BIN file. The highlighed logo is also displayed.

Bootsector only (IP.BIN)
If checked, the IP.BIN will only get put in the bootsector. It will not get added to the filesystem.

Associate Extensions
Opens a very simple window that lets you associate DiscJuggler, CUE and ISO files with BootDreams to Windows Explorer.
